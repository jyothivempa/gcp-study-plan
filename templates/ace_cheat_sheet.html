ACE Cheat Sheet

Essential CLI commands, IAM roles, and key limits to memorize before you walk into the test center.

> gcloud compute

VITAL

Create VM

gcloud compute instances create my-vm --zone=us-central1-a

List Regions

gcloud compute regions list

SSH into VM

gcloud compute ssh my-vm --zone=us-central1-a

> gcloud storage

UPDATED

Make Bucket

gcloud storage buckets create gs://unique-name --location=US

Copy File (Upload)

gcloud storage cp local.txt gs://bucket-name/

Make Public (Risky!)

gcloud storage buckets add-iam-policy-binding gs://bucket --member=allUsers --role=roles/storage.objectViewer

IAM Primitive vs Predefined

Role Type Roles Best Practice
Primitive (Basic) Owner, Editor, Viewer AVOID. Too broad. Only for quick dev tests.
Predefined roles/storage.objectAdmin
roles/compute.instanceAdmin		 PREFERRED. Granular access control.
Custom User created Use when predefined roles don't fit exactly.
