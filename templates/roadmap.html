Associate Cloud Engineer Roadmap

Your structured path from beginner to certified professional in 4 phases.

Exam Overview

Skills Assessed

  • Plan and configure solutions
  • Deploy and implement operations
  • Configure access and security

Register for Exam
  • 1

    Core Fundamentals (Days 1-7)

    Mastering the building blocks: Compute Engine, Virtual Private Cloud (VPC), and Firewall rules. Learning to navigate the console.

  • 2

    Storage & Databases (Days 8-14)

    Distinguishing between options: Cloud Storage (GCS), Cloud SQL, Firestore, and Bigtable. Knowing when to use what.

  • 3

    Security & Operations (Days 15-21)

    Locking it down: IAM roles, Cloud Monitorng, Logging, and Billing controls. Managing access safely.

  • 4

    Advanced & Review (Days 22-30)

    Complex topics: Kubernetes (GKE), App Engine, and Serverless. Followed by intensive practice exams.

  • 5

    Bonus: Advanced Scenarios (Days 31-37)

    The ultimate challenge. 7 days of intensive scenario-based exams to test your architectural limits.

Why this order?

We start with infrastructure (VMs/Network) because it's the foundation. Security comes later once you have resources to secure. Practice exams are last so the knowledge is fresh.
